💡 Conseils pour un bon retour
1
Sois précis. "Ça plante" c'est vague. "Ça plante quand je clique sur X après Y" c'est utile.
2
Joins un screenshot. Une image vaut mille mots, surtout pour les bugs visuels.
3
Explique ce que tu attendais. Parfois ce n'est pas un bug, juste un truc pas clair.
4
Un retour = un problème. Sépare les bugs pour qu'on puisse les traiter indépendamment.
